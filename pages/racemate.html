Racemate is an excercise tracking web app for mobile devices.

Racemate is built on the MERN stack architecture using Apollo Client for elevated state management. MongoDB, Express, React, Node js







Inside the Application

The site can be found here.

Overview

Racemate allows users to create, and track individual exercises. The intent of this application is to give users the ability to track every exercise down to each rep with ease. This includes the ability to easily replicate exercise sets or edit individual reps if desired.
